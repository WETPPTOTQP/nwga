# 舒尔特方格 & 彩色文字游戏

这是一个使用 Python 和 PySide6 构建的练习的一个桌面应用程序，内含两款小游戏：舒尔特方格和彩色文字。它旨在成为一个有趣而简单的大脑训练练习。

## 使用方法

*   打开文件夹dist，双击运行start.exe

## 功能

*   **两款合一的游戏**：
    *   **舒尔特方格**：在方格中按升序尽快找到所有数字。
    *   **彩色文字**：测试您的认知灵活性，根据文字的颜色而非文字本身进行点击。
*   **可自定义的游戏模式**：
    *   为舒尔特方格选择不同的网格大小（3x3、4x4、5x5）。
    *   为两款游戏选择不同的时间限制（10秒、30秒或自定义）。
*   **分数追踪**：每局游戏结束时会显示您的分数。

## 安装

1.  **克隆仓库：**
    ```bash
    git clone https://github.com/WETPPTOTQP/YourRepositoryName.git
    cd YourRepositoryName
    ```


2.  **创建虚拟环境（推荐）：**
    ```bash
    python -m venv .venv
    # 在 Windows 上激活
    .venv\Scripts\activate
    # 在 macOS/Linux 上激活
    # source .venv/bin/activate
    ```

3.  **安装依赖：**
    ```bash
    pip install -r requirements.txt
    ```

## 🎮 如何使用

在项目的根目录中运行以下命令来启动应用程序：

```bash
python main.py
```
